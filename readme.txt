== Installation ==


1. Upload the folder `ooyala-for-wp` folder to your `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Does `Ooyala for WP` support custom video sizes? =

Not yet. You can set default video size settings from WP Admin -> Settings -> Ooyala Video.
